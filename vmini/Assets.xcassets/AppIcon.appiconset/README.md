Drop your macOS app icon PNGs in this folder using these exact filenames:

- icon_16x16.png
- icon_16x16@2x.png
- icon_32x32.png
- icon_32x32@2x.png
- icon_128x128.png
- icon_128x128@2x.png
- icon_256x256.png
- icon_256x256@2x.png
- icon_512x512.png
- icon_512x512@2x.png

If you only have a single master image, make it a sharp 1024x1024 PNG and export the
other sizes from it.
